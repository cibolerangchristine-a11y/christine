Project Chatbot AI: Smart HR Recruiter Assistant
1. Nama Chatbot

TalentMatch AI
2. Use Case

TalentMatch AI adalah chatbot berbasis AI yang membantu tim HR melakukan proses screening kandidat secara otomatis. Chatbot ini dapat:

Menganalisis CV kandidat.
Mencocokkan CV dengan job description.
Memberikan skor kecocokan.
Menjelaskan kelebihan dan kekurangan kandidat.
3. Target Pengguna
HR Recruiter
Talent Acquisition Specialist
Hiring Manager
4. Masalah yang Diselesaikan

Proses membaca CV secara manual memakan waktu lama dan rentan subjektivitas. TalentMatch AI membantu HR mempercepat seleksi awal dengan analisis yang konsisten.

5. Input dan Output
Input
CV kandidat (PDF/DOCX/Teks)
Job Description
Pertanyaan tambahan dari HR
Output
Skor kecocokan (0–100%)
Ringkasan profil kandidat
Kelebihan kandidat
Kekurangan kandidat
6. Model AI yang Digunakan
LLM: OpenAI GPT-4.1 atau GPT-5
NLP untuk ekstraksi skill dan pengalaman
Embedding model untuk semantic similarity
7. Konfigurasi Parameter Model
Parameter	Nilai	Alasan
Temperature	0.2	Output lebih konsisten dan objektif
Max Tokens	1500	Cukup untuk analisis detail
Top_p	0.9	Menjaga relevansi
Frequency Penalty	0.2	Mengurangi pengulangan
Anda adalah HR Recruiter profesional.
Tugas Anda:
1. Analisis CV kandidat.
2. Bandingkan dengan Job Description.
3. Berikan skor kecocokan 0-100.
4. Jelaskan strengths dan weaknesses.
5. Berikan rekomendasi:
   - Lanjut Interview
   - Pertimbangkan
   - Tidak Direkomendasikan
Jawab dalam bahasa Indonesia yang profesional.

Content Creator dengan pengalaman minimal 2 tahun, menguasai Canva, CapCut, SEO, dan copywriting.

Memiliki pengalaman 3 tahun sebagai Social Media Specialist, mahir Canva, CapCut, dan copywriting.
